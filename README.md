# Task-Tracker
A simple CLI based Todo list manager.
## Features
- Add new Task
- Update Task
- List Task by their status
- Delete task by their ID
## Prerequisites
- Node.js in system

## Installation
```bash
git clone https://github.com/Biscute-kk/Task-Tracker.git
```

### Usage
```bash
#navigate to your directory and run
node app.js
```
Run this and follow the interactive i/o system.
